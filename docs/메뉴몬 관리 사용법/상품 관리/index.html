7. 상품 관리

  • 상품 관리는 상품메뉴의 이름이나 가격 등의 정보를 설정하는 기능입니다.
  • 상점 관리 화면에서 해당 업장 페이지로 들어갑니다.
  • 메뉴 관리 카테고리에서 상품 관리 버튼을 클릭합니다.
  • 상품 목록 화면에서 좌측 상단의 추가 혹은 우측 하단의 + 버튼을 누릅니다.
    • 신규 상품을 생성 할 수 있는 화면에서 새로 추가할 상의 이름과 가격, 상품 이미지, 수량 적용 여부, 수량 적용시 구매가능한 최소 수량과 최대 수량, 수량 표기 방법 등의 기본 정보들을 세팅 할 수 있습니다.
      • 상품 이미지는 이미지 없음을 클릭하여, 이미지 관리에서 미리 등록했던 이미지를 불러와 적용할 수 있습니다.
    • 화면 하단부의 상세 설정 기능을 이용하면 해당 상품에 대해 더 세부적인 설정을 적용할 수 있습니다.
      • 상세 설정의 기본 정보 메뉴에서는 상품의 이름과 설명, 밴드 플래그, 포장 가능 여부를 설정 할 수 있습니다.
      • 프린터 출력시 이름은 POS를 연동하지 않는 업장에서 주문앱에 표시되는 상품명과 주방으로 들어가는 주문서에 출력되는 상품명을 따로 설정하길 윈할때 사용하는 기능입니다. (예시. 주문앱에는 '세상에서 가장 맛있는 짱짱라면' 이란 상품명이, 주방으로 가는 주문서에는 '라면' 이라고 표시되게 할 수 있음)
      • 밴드 플래그는 상품 이미지에 BEST, HOT, SPECIAL 등의 아이콘이 함께 노출되어 해당 상품이 돋보일 수 있도록 하는 기능입니다.
      • ~포장 가능 여부는 사용하지 않습니다~
      • 가격 정보 메뉴에서는 상품의 가격과 면세가격, 부가세, 봉사료 등의 상세 내역을 설정 할 수 있습니다.
      • 수량 정보 메뉴에서는 상품의 수량 적용 여부와 수량 적용시 구매가능한 최소 수량과 최대 수량, 수량 표기 방법을 설정 할 수 있습니다.
      • 노출 설정 에서는 판매 가능 시간, 판매 가능 기간 설정 사용 유무를 설정할 수 있습니다.
      • 노출 설정 에서 ~숨김 여부와 판매중지 여부 기능은 사용하지 않습니다~
      • 옵션 그룹 목록에서는 상품에 해당하는 옵션 그룹을 추가/삭제하는 기능입니다. 추가/삭제버튼을 통해, 해당 상품을 미리 설정해둔 옵션 그룹의 구성 상품으로 추가 또는 삭제 하고, 드래그 앤 드롭을 통해 정렬 순서를 지정할 수 있습니다.
      • 주문시 출력 프린터 선택 메뉴에서는 해당 상품의 주문서가 업장에 세팅 되어있는 프린터 중 어떤 프린터에서 출력되게 할지 세팅 할 수 있습니다.(예. 음료 상품은 카운터에 있는 프린터로, 육류 상품은 주방에 있는 프린터로 출력되게 끔 상품별로 각각 설정 할 수 있음) POS 연동 업장에서는 사용하지 않습니다.
      • POS 연동 업장에서는 POS 연동 설정 메뉴를 통해 해당 상품을 POS와 연동해주어야 합니다.
      • POS 연동 업장이라면 상품으로 취급을 활성화 해줍니다.
      • ~상위 상품으로 대체는 상품에서는 사용하지 않습니다.~
      • ~포스 카테고리는 따로 입력하지 않아도 됩니다.(현재 사용하지 않는 기능)~
      • 주문앱의 상품과 POS의 상품을 매칭하기 위해선 POS의 상품 리스트에서 매칭하고자 하는 상품의 POS 코드를 확인 한 뒤, 포스 상품코드란에 꼭 입력해줘야 합니다.
      • 핫키 사용여부(Oasis POS전용)은 점주가 메뉴몬 관리앱에서 가격을 변경하고 손님이 그 변경된 가격으로 주문을 했을시, POS에서도 바뀐 가격으로 주문이 입력되게 끔 하는 기능입니다. POS에서 상품 가격을 변경하기 귀찮을때 사용하는 기능이지만, 헷갈릴 수 있으니 특별한 경우가 아니면 사용하지 않는 것을 권장합니다.
      • ~상품 이름 대체 방법 기능은 옵션에서만 사용하고 상품에서는 사용하지 않습니다~
      • 추가 설정 메뉴의 주문시 정렬 순서는 주문서 출력시, 주문서에 표시되는 메뉴들의 정렬 순서를 정할 수 있는 기능입니다. 손님이 주문앱에서 어떤 순서로 상품을 장바구니에 담았는지와 관계없이 주문서에는 주문시 정렬 순서로 설정된 순서대로 메뉴가 표기되어 출력됩니다. 기본 설정은 "50" 으로 입력되어 정렬 순서가 없고, 정렬 순서를 요청하는 업장에서만 사용하는 기능입니다. 예를들어 "햄버거" 의 정렬순서가 "1" 이고, "감자튀김" 의 정렬순서가 "2" 고, "콜라" 의 정렬순서가 "3" 으로 설정돼 있다면, 고객이 장바구니에 "콜라" -> "햄버거" -> 감자튀김" 순서로 상품을 담아도, 주방에서 출력되는 주문서에는 "햄버거" -> "감자튀김" -> "콜라" 순서대로 표시됩니다.
      • 기타 설정 메뉴의 메모기능과 별칭기능은 메뉴몬 관리페이지 작업자가 작업에 용이하도록 특이사항을 적어두는 기능입니다. 예를들어 "A세트" 라는 상품이 런치 메뉴로 있을 수 있고 디너 메뉴로도 있을수 있는데 이럴 때 헷갈리지 않게 하기 위해서 별칭에 "A세트 런치" 혹은 "A세트 디너" 와 같이 적어두면 추후 작업시 간편하게 구분 할 수 있습니다.(별칭은 옵션목록에 별칭:으로 표시되지만 메모는 해당 옵션 상품의 상세 설정에서만 확인가능)
  • 모든 변경 사항은 관리 페이지 하단의 저장하기버튼을 누른다음 주문/관리앱에서 앱 업데이트까지 진행해야 비로소 적용됩니다.
