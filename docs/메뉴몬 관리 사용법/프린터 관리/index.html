13. 프린터 연동 설정

  • 상점 관리 화면에서 해당 업장 페이지로 들어갑니다.
  • 연동/통합 관리 카테고리에서 프린터 관리 버튼을 클릭합니다.
  • 프린터 관리는 메뉴몬 관리 테블릿과 연동하는 프린트의 설정을 관리하는 것입니다. POS를 연동하여 사용하는 업장의 경우, POS기에서 프린터 설정을 해야합니다.
  • 프린터 연동 우측 추가 아이콘을 클릭합니다.
  • 프린터 정보, 프린터 아이디는 1,2,3... 순서로 입력합니다. 아이디는 중복된 이름으로 입력하면 안됩니다.
  • 프린터 이름에는 프린터를 구분할 수 있도록 이름을 입력합니다. 예를 들어 주방에서 사용하는 경우 주방용 카운터에서 사용하는 경우 카운터용 이라고 입력합니다.
  • 프린터 유형 선택에서 해당 프린터의 용도에 맞게 주문서, 영수증 중 원하는 유형을 선택할 수 있습니다. (중복 선택 가능)
  • 아이피 설정, 아이피 타입 선택에서 직접입력, 이더넷, 와이파이 중 해당하는 아이피를 선택합니다. IP 직접 입력하기에 IP를 입력합니다.
  • 포트 설정 , 프린터 선택에서 해당 하는 프린터를 설정합니다. 삼포스 를 프린터의 경우 랜선으로 연결하면 port 선택에서 6001을 와이파이로 연결하면 port 선택에서 9100을 선택합니다. 나머지 프린터의 경우 port 선택에서 9100을 선택합니다. 직접 입력하기를 선택하여 입력할 수도 있습니다.
  • 프린터 모델명에 해당 프린터 모델명을 입력합니다.
  • 저장을 선택하여 해당 프린터가 추가된 것을 확인할 수 있습니다.
  • 저장하기를 선택해야 설정한 것들이 적용됩니다.
  • 상품 관리에서 상품마다 주문시 출력 프린터 선택을 통해 등록한 프린트와 연동해주어야합니다.
  • 앱 설정, 관리 앱 설정에서 어드민으로 프린트 출력 사용 유무토글을 활성화해주어야합니다.
