11. POS 연동 설정

  • 상점 관리 화면에서 해당 업장 페이지로 들어갑니다.
  • 연동/통합 관리 카테고리에서 POS 연동 관리 버튼을 클릭합니다.
  • 포스 정보 메뉴에서는 해당 업장의 POS와 메뉴몬 연동하기 위한 설정을 할 수 있습니다.
    • POS와 연동하기 위해선 먼저, 포스 연동 사용유무 토글을 활성화 합니다.
    • 메뉴몬과 연동 가능한 POS 리스트 중, 해당 업장에서 사용하는 POS를 선택합니다.
    • 현재 메뉴몬은 KS_POS, EASY_POS, OASIS_POS, UNION_POS, KIS_POS, M_POS, ANNECY_POS와 연동할 수 있습니다.(2023년 6월 2일 기준)
    • KS_POS 연동 하기
      • UNION_POS를 제외한 나머지 모든 포스들을 연동하기 위해선 공통적으로 포스 IP와 포스 PORT값을 꼭 입력해줘야합니다. (IP는 해당 업장 포스기에서 확인 가능하고 포트는 입력란을 클릭하면 표시되는 값으로 입력하면 됨)
      • 주문서 출력유무는 메뉴몬 주문앱에서 발생한 주문이 포스로 넘어왔을때 주문서를 출력할지 말지 여부를 설정하는 기능입니다.(계산서 , 전표 출력 유무 역시 마찬가지 기능입니다.)
      • 계산완료 후 테이블 점유해제(선결제 Only 매장에서만 사용가능) 는 주문앱에서 선결제 주문이 이뤄짐과 동시에 포스기에서 자동으로 정산처리(자리비움)까지 진행되게 하는 기능 입니다. 점주분들의 요청이 있는 경우에 사용하시면 됩니다.
      • 영수증 출력유무(계산완료 후 테이블 점유해제 기능 활성화 일 때만 사용 가능) 는 선결제가 완료 됐을때 포스에서 자동으로 영수증을 출력하는 기능입니다. 다만, 해당기능은 KS포스 특성상 주문완료로부터 영수증 출력까지 약 30초가 소요되기때문에 되도록이면 사용하지 말고, 영수증은 요청하는 손님의 요구가 있을때마다 포스에서 직접 발급하는 것을 권장합니다.
    • EASY_POS 연동 하기 (이지포스는 포스기 특성상 선불형과 후불형을 동시에 사용할 수 없습니다)
      • 포스 IP와 포스 PORT 값을 입력해줍니다.
      • 본부코드와 매장코드는 해당 매장의 밴대리점에 문의해야 합니다.
      • 결제전용 포스번호는 이지포스 매장에서 메뉴몬을 선불로 사용할때 필요한 값으로서, 20 을 입력하면 됩니다. 이는 메인포스와 주문앱으로부터 전달되는 데이터가 충돌되지 않게끔 포스에 가상의 번호(20번) 을 부여하는 것입니다. 해당 설정을 위해서는 포스 대리점에게 연락하여 포스번호를 20번까지 늘려달라고 해야합니다.(실제로 포스를 20개 쓰는 것이 아니라 전상상의 포스번호를 늘리는 것이라고 설명해줘야함)
      • 영수증번호는 00시00분을 기준으로 포스에서 출력되는 영수증 번호를 설정 할 수 있는 기능입니다. 예를들어 영수증번호를 "001" 로 설정해두면 해당 업장에서 00시 00분 이후 "1번테이블" 에서의 첫 주문 영수증이 "1(테이블번호)-001(영수증번호)" 로 출력됩니다. "2번테이블" 에서의 첫 주문은 "2(테이블번호)-001(영수증번호)" 로 출력되고, "1번테이블" 에서의 두번째 주문은 "1(테이블번호)-002(영수증번호)" 로 출력됩니다. 예시에 표기 된 자릿수를 지켜서 입력해주셔야합니다.(001로 입력)
    • OASIS_POS 연동 하기
      • OASIS_POS 는 포스 IP와 포스 PORT 값만 입력해주면 됩니다.
    • UNION_POS 연동 하기
       * `UNION_POS`와 연동하기 위해서 `유니온 연동 API-KEY`를 입력해야합니다. 해당 연동키는 해당 대리점 또는 유니온포스 고객센터를 통해 확보하면 됩니다.
       * `유니온포스 시퀀스`는 동일한 사업자 번호로 여러개의 매장을 운영할 때 사용하는 기능입니다.
       * `유니온포스 시퀀스`를 사용하기 위해선 토글을 활성화하고 아래에 있는 `유니온포스 시퀀스` 입력창에 해당 값을 입력해야합니다.**(유니온 ASP 페이지에서 사업자 등록번호 뒤에 적혀있는 값)**
       * 예를들어 하나의 사업자등록증으로 **"1호점"** , **"2호점"** 을 운영한다고 하면 메뉴몬 어드민 페이지에서 **"1, 2호점"** 각각의 상점페이지를 만들고 **"2호점"** 상점 페이지에서는 **"2호점"** 의 `유니온포스 시퀀스`값을 넣어주셔야합니다.(**"1호점"** 은 따로 입력 필요없고 **"2호점"** 부터)
    • KIS_OKPOS 연동 하기
      • 포스 IP와 포스 PORT 값을 입력합니다.
      • 영수증번호는 00시00분을 기준으로 포스에서 출력되는 영수증 번호를 설정 할 수 있는 기능입니다. 예를들어 영수증번호를 "001" 로 설정해두면 해당 업장에서 00시 00분 이후 "1번테이블" 에서의 첫 주문 영수증이 "1(테이블번호)-001(영수증번호)" 로 출력됩니다. "2번테이블" 에서의 첫 주문은 "2(테이블번호)-001(영수증번호)" 로 출력되고, "1번테이블" 에서의 두번째 주문은 "1(테이블번호)-002(영수증번호)" 로 출력됩니다. 예시에 표기 된 자릿수를 지켜서 입력해주셔야합니다.(0001로, 네자리로 입력)
    • M_POS 연동하기엠포스는 포스기 특성상 선불형과 후불형을 동시에 사용할 수 없습니다
        *  `주문서 출력유무`는 메뉴몬 주문앱에서 발생한 주문이 포스로 넘어왔을때 주문서를 출력할지 말지 여부를 설정하는 기능입니다.(`계산서, 전표 출력 유무` 역시 마찬가지 기능입니다.)
        * `영수증번호`는 00시00분을 기준으로 포스에서 출력되는 영수증 번호를 설정 할 수 있는 기능입니다. 예를들어 `영수증번호`를 **"001"** 로 설정해두면 해당 업장에서 00시 00분 이후 **"1번테이블"** 에서의 첫 주문 영수증이 **"1(테이블번호)-001(영수증번호)"** 로 출력됩니다. **"2번테이블"** 에서의 첫 주문은 **"2(테이블번호)-001(영수증번호)"** 로 출력되고, **"1번테이블"** 에서의 두번째 주문은 **"1(테이블번호)-002(영수증번호)"** 로 출력됩니다. 예시에 표기 된 자릿수를 지켜서 입력해주셔야합니다. (001로 입력)
    • ANNECY_POS 연동하기
      *  `POS URL`에 해당 POS가 연결된 IP 주소를 입력합니다.(**http://ip주소:3443** 와 같이 입력합니다)
      *  `매장코드`는 해당 매장의 밴대리점에 문의해야 합니다.
  • 해당 정보들을 입력한 후 저장하기를 누르면 포스 연동 작업이 완료됩니다.
