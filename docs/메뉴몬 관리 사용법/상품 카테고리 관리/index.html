8. 상품 카테고리 관리

  • 상품 카테고리 관리에서는 개별 상품들을 그룹화 할 수 있습니다.
  • 상점 관리 화면에서 해당 업장 페이지로 들어갑니다.
  • 메뉴 관리 카테고리에서 상품 카테고리 관리 버튼을 클릭합니다.
  • 옵션 그룹 목록 화면에서 좌측상단의 추가 혹은 우측 하단의 +버튼을 누릅니다.
    • 신규 카테고리를 생성 할 수 있는 화면에서 새로 추가할 카테고리의 이름을 세팅 할 수 있습니다. (ex. "면" , "밥" , "사이드 메뉴" 등)
    • 카테고리 정렬 순서를 통해 테블릿 메뉴판에서의 정렬 순서를 정할 수 있습니다. 기본 세팅은 "0" 으로 되어있으며, 10단위의 세자리 수로 입력해야합니다. 예를 들어 가장 상단에 노출 돼야하는 카테고리부터 "010" , 그 다음 순서는 "020" , "030" , "040"... 와 같은 방식입니다.
    • 화면 하단부의 상세 설정 기능을 이용하면 해당 상품 카테고리에 대해 더 세부적인 설정을 적용할 수 있습니다.
        * ~`설명`의 경우 상품 기준으로 설정해야 표시 됩니다. = 상품 카테고리 관리에선 신경 안써도 됨~ 
        * ~`프린터 출력시 이름`은 POS를 연동하지 않는 업장에서 주문앱에 표시되는 상품명과 주방으로 들어가는 주문서에 출력되는 상품명을 따로 설정하길 윈할때 사용하는 기능입니다.= 상품 카테고리 관리에선 신경 안써도 됨~
        * ~`밴드 플래그` 기능과 `포장 가능 여부` 기능은 현재 상품 카테고리에서 작동하지 않고, `상품 관리`메뉴에서 상품 기준으로 설정해야 표시 됩니다 = 상품 카테고리 관리에선 신경 안써도 됨~
        ![015](./images/상품카테고리관리/015.png)
       * `카테고리 상품 목록` 메뉴에서는 해당 상품 카테고리를 구성하고 있는 상품을 추가/삭제 할 수 있습니다. `추가/삭제`버튼을 통해, 미리 등록한 상품을 해당 상품 카테고리에 등록하고, 드래그 앤 드롭을 통해 정렬 순서를 지정할 수 있습니다.
       ![016](./images/상품카테고리관리/016.png)
       * `노출 설정`의 `숨김 여부`를 통해 카테고리를 노출 혹은 숨길 수 있습니다.
       * ~`노출 설정의` `판매 중지 여부`, `판매 가능 시간`, `판매 가능 기간`은 상품 카테고리 관리에서 신경 안써도 됨~
       ![017](./images/상품카테고리관리/017.png)
       * `카테고리 타입`은 해당 카테고리를 일반적인 메뉴 구분 외에, 특수한 용도로 사용하고 싶을때 사용하는 기능입니다.
       * `카테고리 타입` 중 `기본`을 선택하면 해당 카테고리가 `면류`, `밥류`, `사이드 메뉴`와 같이 일반적인 메뉴 카테고리로 세팅됩니다.
       ![018](./images/상품카테고리관리/018.png)
       * `카테고리 타입` 중 `인원 수 선택`은 손님이 해당 카테고리를 눌렀을때 메뉴 목록이 뜨는게 아니라 인원 수를 선택하는 화면이 뜨게하는 기능 입니다. `인원 수 선택`을 선택하면 `상품 연결하기`를 통해 상품을 연결 해줘야 하는데 `상품 관리` 메뉴에서 `인원 수`라는 상품을 따로 생성한 다음 해당 상품을 연결해줘야 인원 수 선택 기능이 정상적으로 작동합니다. 테이블마다 인원수를 확인해야하는 매장이나, 상차림비를 인원수마다 받는 업장에서 사용합니다. 상차림비를 인원수마다 받는 업장의 경우, `인원 수`상품에 인당 상차림비 가격을 입력해야합니다. 
       ![019](./images/상품카테고리관리/019.png)
       * ~`카테고리 타입` 중 `상품설명 카테고리`는 손님이 해당 카테고리를 눌렀을때 메뉴 목록이나 인원 수 선택 화면이 아니라 텍스트 정보를 노출시키고 싶을 때 사용합니다. "일본 라멘 더 맛있게 먹는 법" 등 매장에서 손님에게 안내하고 싶은 내용이 많을때 이 기능을 사용합니다. 이 기능은 `이용안내` 기능으로 갈음 할 수 있으므로, 사용하지 않는것을 권장합니다.~
       ![021](./images/상품카테고리관리/021.png)
  • 모든 변경 사항은 관리 페이지 하단의 저장하기버튼을 누른다음 주문/관리앱에서 앱 업데이트까지 진행해야 비로소 적용됩니다.
