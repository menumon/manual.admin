3. 앱 설정

  • 상점 관리 화면에서 해당 업장 페이지로 들어갑니다.

  • 태블릿 / 앱 관리 카테고리에서 앱 설정 버튼을 클릭합니다.

  • 앱 설정 화면은 크게 공통 설정, 주문앱 설정, 관리앱 설정의 세 파트로 구분 돼 있습니다.

  • 먼저 공통 설정입니다.

    • 언어 설정 메뉴에서는 사용할 언어의 종류를 설정 할 수 있습니다.
      • 다만 메뉴명 등이 자동 번역 되는 것이 아니라 직접 해당 언어로 메뉴 세팅을 해주셔야 합니다.
      • 다국어 설정 사용유무 스위치를 통해 여러 언어를 동시에 노출 시키는 것도 가능합니다.
      • 기본 언어 선택을 통해 메뉴몬 화면에 표시 될 기본 언어의 종류를 선택 할 수 있습니다.
    • 세금 / 봉사료 메뉴에서는 '부가세' 와 '봉사료' 의 포함 유무와 '봉사료 요율' 을 설정 할 수 있습니다.
      • 기본적으로는 '부가세 포함' , '봉사료 없음'으로 설정하면 됩니다.

  • 다음은 주문앱 설정입니다.

    • 공지 정보 메뉴에서는 주문 태블릿 상단에 표시 되는 공지 사항 내용을 설정 할 수 있습니다

    • 광고시작까지 대기시간 설정 메뉴에서는 태블릿 조작 없이 얼마의 시간이 지나야 화면에 광고가 표시 되는지 설정 할 수 있습니다.

    • UI 설정 메뉴에서는 주문앱의 UI(User Interaface) 스타일을 변경 할 수 있습니다.

  • 결제 방법 설정 메뉴에서는 해당 업장에서 어떤 방식으로 결제를 진행할지 설정 할 수 있습니다.

    • 선불(카드) 기능 사용 유무 를 활성화하면 태블릿에 카드 단말기를 설치하여 주문과 동시에 결제를 진행하는 선불 기능을 사용할 수 있습니다.
    • `후불 기능 사용 유무'를 활성화 하면 손님이 식사 후 매장을 나갈 때 카운터에서 결제를 진행할 수 있게 선 결제 없이 음식을 주문 할 수 있게 됩니다.
    • 선불 기능과 후불 기능을 모두 활성화 하면 손님이 주문을 할 때, 선불로 진행할 지 혹은 후불로 진행할지 선택할 수 있게 됩니다.
    • 선불 기능과 후불 기능 각각 활성화 시, 주문앱에 노출되는 버튼 문구를 자유롭게 설정 할 수 있는 칸이 나타납니다.(각각 비교 이미지 삽입)
    • 상품 가격 부가세 표시, 장바구니 주문 미리보기에 부가세 상세 미표시 기능을 통해서 주문앱에 부가세, 봉사료 등 가격에 대한 상세 정보 표시 유무를 설정할 수 있습니다.
    • 결제 전 "주문하시겠습니까?"팝업 미사용을 활성화하면 손님이 주문앱에서 최종적으로 주문하기를 터치했을 때 "주문하시겠습니까?"라는 별도 팝업이 없이 바로 주문이 접수 됩니다.
    • ~더치페이 기능 사용 유무는 현재 개발 중인 기능이니 별도 안내가 없을 때까지 사용하지 말아주십시오.~

  • 선불 결제 버튼 비활성 시간 설정 기능은 시간대 / 요일 별로 선불 기능을 사용했다 안했다 하는 매장일 경우(ex. 점심 영업 시간에는 선불 기능 사용, 저녁에는 후불만 가능) 수동으로 매번 바꿔주지 않아도 미리 세팅해 둔 요일과 시간대에 따라 자동으로 선불 기능이 활성/비활성화 되게끔 하는 기능입니다.

  • 후불 주문 버튼 비활성 시간 기능 역시 위와 마찬가지로 자동으로 후불 기능이 활성/비활성화 되게끔 하는 기능입니다.

  • 주문시 테이블 선택기능을 활성화 하면 여러 테이블에 앉은 손님들이 키오스크 처럼 한대의 주문 태블릿으로 각각 테이블 별로 주문을 할수 있게 됩니다.(태블릿 메뉴판이 테이블 마다 하나씩 필요하지 않은 업장일 경우 사용)

  • 상차림비 사용설정 기능을 활성화 하면, 첫 주문시 상차림 비를 적용할 수 있습니다. 테이블당 상차림비를 적용하는 업장에 사용을 추천합니다.

    • 상차림비 상품 연결을 통해 상차림비라는 메뉴를 연결합니다. (상품 관리메뉴얼을 참고하여, 상차림비 라는 메뉴를 먼저 생성한 이후 연결해야 함) 상차림비는 첫 주문시 손님이 아니라, 직원이 직접 테블릿 메뉴판의 설정에서 상차림비 적용하기를 적용해야 합니다. (인원수대로 상차림비가 적용되는 매장의 경우 상차림비 사용 설정기능을 사용하지 않고, 상품 카테고리 관리의 카테고리 타입을 통해 기능을 설정합니다. 메뉴얼 상품 카테고리 관리참조)

  • 포장 정보 기능은 손님이 주문 태블릿으로 포장 주문을 했을 때, POS와 관리앱으로 해당 주문 정보를 전달하는 기능입니다.

    • 포장 정보 기능은 해당 버튼 상단에 기재 되어있는 POS사와는 연동이 되고, 연동이 어려운 경우는 관리 테블릿에서 포장여부를 확인할 수 있습니다. 포장 불가 상품을 따로 설정하기는 어려우니, 포장 불가 상품이 따로 있는 경우 이름에 "포장 불가" 라고 표시해두는 것이 좋습니다.

  • 테이블 당 최대 매장 이용시간에 제한이 있는 업장일 경우 식사 시간 제한 정보 기능을 이용해 손님들의 식사 시간을 제한 할 수 있습니다.

    • 시간 제한 사용 유무 버튼을 활성화 하여 손님이 식사 할 수 있는 최대 시간을 분 단위로 입력합니다.
    • 그 다음 시간 제한 상품 연결 버튼을 클릭하여 시간 제한이라는 메뉴를 연결합니다. (상품 관리메뉴얼을 참고하여, 시간 제한이 라는 메뉴를 먼저 생성한 이후 연결해야 함)
    • POS 연동 매장에서 해당 기능을 이용하기 위해선, POS에도 시간 제한 상품이 있어야 합니다.(시간 종료 알림 용도)
    • 타이머 설정은, 손님이 착석한 이후, 직원이 테블릿 메뉴판의 설정에서 타이머 시작을 선택하여 적용할 수 있습니다. 테블릿 메뉴판 상단에서 남은 시간을 확인할 수 있습니다. 제한된 시간이 끝나면, 시간이 종료되었다는 메시지가 테블릿 메뉴판에 표시 되고 주문서에도 함께 출력됩니다.

  • 이용안내 이미지 선택 기능을 사용하면 주문앱 좌측 하단에 이용안내 버튼이 생기며, 매장 별 이용안내 이미지를 노출 시킬 수 있습니다.

  • 다음은 기타 앱 설정입니다.

    • 주문내역 보기 버튼 사용유무 기능을 통해, 주문앱에서 손님이 주문내역을 확인 할 수 있게 할지 말지, 주문내역을 최근 몇 건까지 노출시킬 것인지 설정 할 수 있습니다
    • 주문시 상품 정렬 사용유무 기능을 활성화 하면 주문이 주방으로 들어갈 때, 손님이 주문앱에서 상품을 담은 순서가 아니라 매장에서 미리 설정한 순서에 따라 정렬되어 표시 됩니다.
    • 폰트 사이즈는 크게 상세 화면 폰트 크기, 카테고리 폰트 크기, 메뉴 리스트 폰트 크기, 네비게이션바 높이로 구분 돼있습니다.(각각 영역 표시해서 이미지 첨부)
    • 직원호출 글자 보이기 기능을 활성화 하면 주문앱 좌측 하단에 서비스 요청이라는 버튼이 생깁니다.
    • 서비스 요청 항목들은 해당 업체 페이지의 서비스 호출 관리 메뉴에서 추가/삭제 할 수 있습니다.
    • 선불 매장인지 후불 매장인지에 따라 태블릿 일반 모드 / 키오스크 모드 기능을 통해 모드를 변경해줘야합니다. 키오스크 모드일때 카드 리더기가 인식되지 않는 현상으로 인해, 선불기능을 사용하는 매장은 일반모드를 사용하고, 후불 기능만 사용하는 매장은 키오스크 모드 사용을 권장합니다.(키오스크 모드는 메뉴몬 바깥의 바탕화면으로 못나가게 막아둠)

  • 마지막으로 관리앱 설정입니다.

    • 관리앱 설정에서는 어드민으로 프린트 출력 사용유무, 집계현황 사용유무, 결제 취소 기능 사용유무, 주문 테이블에 상품준비 완료 알림 사용유무, 어드민 기기 여러대 사용유무를 설정할 수 있습니다.
    • 기타 설정에서 관리앱도 일반 모드와 키오스크 모드를 선택할 수 있는데요, 바탕화면으로 나가지 못하게 하는 키오스크 모드의 사용을 권장하지만 결제 취소를 위해 어드민 태블릿에 카드 리더기를 연결해야 할 경우 카드 디러기 인식을 위해 일반 모드로 변경해주셔야 합니다.

  • 모든 변경 사항은 관리 페이지 하단의 저장하기버튼을 누른다음 주문/관리앱에서 앱 업데이트까지 진행해야 비로소 적용됩니다.
